Texture AutoLookup (TAL) for Windows
version 4.2

Purpose of program:
A Win95/2000/NT/XP program to determine the
soil texture class based on several soil
classification schemes including the USDA (U.S.
Dept. of Agric.), UK, Canadian, and
International schemes. In addition, TAL now
supports custom (user-defined) classification
schemes, and charting capabilities. For more
information and the latest updates, please go
to: www.agri.upm.edu.my/~chris/tal.

Installation:
Run Setup.exe
