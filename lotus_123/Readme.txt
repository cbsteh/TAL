=======================================================================
3.0   Installation


   Installation
   ------------

   1. Create a directory called "Texture" at the root directory
      of your hard disk (e.g. c:\Texture if your hard disk is the C
      drive, or d:\Texture if your hard disk is the D drive). 

   2. Copy all the files as listed in section 2.0 into the Texture
      directory.

   3. Add "Texture" in the PATH statement. To do this, open the
      AUTOEXEC.BAT file (kept in the root directory, that is, C:\ or
      D:\), and add the following line:

                    PATH = %PATH%;C:\TEXTURE

      if you had previously created the "Texture" directory in your
      C drive, OR add this line:
 
                    PATH = %PATH%;D:\TEXTURE

      if you had previously created the "Texture" directory in your
      D drive.


=======================================================================
4.0   Quick Start


   Launch Lotus 1-2-3 as usual and open the file "TAL_123.WK4" as you
would do with any other Lotus 1-2-3 files.

   Once the TAL code file is opened, the menu "TEXTURE" will
automatically appear at the spreadsheet's main menu bar. All TAL commands
are available from this Texture menu.

   For complete instructions of how to use TAL, choose Help from the
Texture menu. Please contact me (at my address or by e-mail) if you have
any comments or queries. I would be glad to help. 

   Enjoy TAL!
